## Method

Synthetic Chromium run against local Vite preview servers. Metric is browser-side ready time: SvelteKit hard navigations use the document-shell marker after the initial head is parsed, while still waiting for each page readiness selector; SPA/hash transitions mark before the navigation action and measure until the target section/page is present or active. Browser cache stays enabled after warmups.
