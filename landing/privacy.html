Politique de confidentialité

Dernière mise à jour : Mars 2026

MissionPulse est une extension de navigateur conçue avec la confidentialité comme principe fondateur. Cette politique de confidentialité explique comment nous gérons vos données.

Principe fondamental

Toutes vos données restent sur votre machine. MissionPulse est une application 100% locale. Aucune donnée personnelle n'est transmise à nos serveurs, car nous n'en avons pas. Il n'y a pas de backend, pas de base de données distante, pas de tracking, pas d'analytiques.

Données stockées localement

Les données suivantes sont stockées exclusivement sur votre ordinateur :

  • Profil utilisateur — Vos compétences, TJM cible, localisation, préférences de télétravail et niveau de seniorité que vous configurez lors de l'onboarding.
  • Missions scrapées — Les missions récupérées depuis Free-Work, Malt et Comet sont stockées dans IndexedDB sur votre machine.
  • États des connecteurs — Le statut de vos connexions aux différentes plateformes.
  • Cache de scoring sémantique — Les scores de pertinence calculés par Gemini Nano sont mis en cache localement pour optimiser les performances.
  • Préférences de l'extension — Vos paramètres de notification et de scan automatique.

Utilisation de Gemini Nano

MissionPulse utilise Gemini Nano, l'IA intégrée de Chrome, pour le scoring sémantique des missions. Cette IA fonctionne entièrement sur votre ordinateur (on-device). Aucune donnée n'est envoyée à Google ou à des serveurs externes pour cette fonctionnalité.

Accès aux plateformes tierces

MissionPulse accède aux sites Free-Work, Malt et Comet pour scrapper les missions disponibles. Ces accès sont effectués via votre session navigateur existante — vous devez être connecté à ces plateformes. Nous ne stockons jamais vos identifiants de connexion.

Les conditions d'utilisation de ces plateformes tierces s'appliquent à votre utilisation de leurs services. MissionPulse n'est pas affilié à ces plateformes.

Aucun tracking ni analytique

MissionPulse ne collecte aucune donnée d'usage. Nous n'utilisons pas :

  • Google Analytics ou tout autre service d'analytique
  • Cookies de tracking
  • Fingerprinting
  • Télémétrie
  • Rapports d'erreur automatisés

Communications

Nous ne vous enverrons jamais d'emails marketing ou de notifications push commerciales. Les seules notifications que vous recevrez sont celles que vous avez explicitement configurées pour être alerté des nouvelles missions.

Code source ouvert

MissionPulse est un projet open source. Vous pouvez vérifier par vous-même comment l'extension fonctionne en consultant notre code source sur GitHub. La transparence est notre meilleure garantie de respect de votre vie privée.

Suppression de vos données

Toutes vos données étant stockées localement, vous avez le contrôle total. Pour supprimer toutes vos données :

  • Désinstallez l'extension depuis Chrome (chrome://extensions)
  • Les données IndexedDB et chrome.storage associées seront automatiquement supprimées

Mises à jour de cette politique

Si nous modifions cette politique de confidentialité, nous en informerons les utilisateurs via une notification dans l'extension. Les changements significatifs nécessiteront votre consentement explicite.

Contact

Pour toute question concernant cette politique de confidentialité ou vos données, vous pouvez nous contacter à : privacy@missionpulse.app
