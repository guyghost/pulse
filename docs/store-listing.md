## Description detaillee

**MissionPulse** est une extension Chrome pour les freelances qui centralise les missions de plusieurs plateformes dans un panneau lateral unique.

### Fonctionnalites

- **Scraping automatique** — Scanne en arriere-plan Free-Work, Malt et Comet a intervalles reguliers. Aucune action manuelle requise.
- **Score de pertinence** — Chaque mission recoit un score base sur vos competences, votre TJM cible et vos preferences. Les missions les plus pertinentes remontent en tete.
- **Analyse TJM** — Exploite l'API Anthropic pour analyser le taux journalier moyen et le comparer a votre cible.
- **Deduplication intelligente** — Detecte et fusionne les missions publiees sur plusieurs plateformes simultanement.
- **Notifications** — Recevez une alerte des qu'une mission correspondant a votre profil est detectee.
- **100 % local** — Toutes vos donnees restent sur votre machine. Aucun serveur externe, aucun tracking.

### Comment ca marche

1. **Installez** l'extension depuis le Chrome Web Store.
2. **Configurez** votre profil : competences, TJM cible, preferences.
3. **Connectez-vous** a vos plateformes (Free-Work, Malt, Comet) dans des onglets separes.
4. **Ouvrez le panneau lateral** — MissionPulse scanne automatiquement et affiche vos missions triees par pertinence.

### Vie privee

MissionPulse ne collecte aucune donnee personnelle. Tout est stocke localement via `chrome.storage.local` et IndexedDB. Votre cle API Anthropic reste sur votre machine et n'est jamais partagee. Consultez notre politique de confidentialite pour plus de details.
